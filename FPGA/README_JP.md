# 内容：FPGA設計データ

## 概要
- Sipeed Tang Mega 138k用の設計データです。

## フォルダ構成

| フォルダ名　　　　　　　| 内容                                  |
|------------|--------------------------------------|
| archive    | デザインのアーカイブファイル              | 
| impl       | 現状は空フォルダ                        | 
| src        | FPGA設計コード                         |

## インジケーター（ダミー）
- tang mega 138k ボードのPMOD2コネクターにPMOD LCDを接続するとインジケーターが表示されます。
- 現状ではダミー表示です。
